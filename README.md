# New Super Mario Bros. DELUXE

> A modernized, cross-platform remake of the classic *New Super Mario Bros. DS*.

Welcome to the official repository for **New Super Mario Bros. DELUXE**. This project aims to bring the classic DS experience to all modern platforms with fresh enhancements, while staying true to the original feel of the game.

## ✨ Features

* **Cross-Platform Play:** Playable on Windows, macOS, Linux, Android and more.
* **Remastered Soundtrack:** [OST](https://www.youtube.com/playlist?list=PLjGZNqL8Ia9DhdRolzPFBe7WxjukTGdz5) Enjoy the classic tunes, completely remastered for high-quality audio.
* **Updated Graphics:** Choose between modern widescreen or classic dual-screen layouts. Experience the Mushroom Kingdom like never before with high-resolution 3D models and subtle visual tweaks that polish the overall look.
* **Customization:** Various new settings and personalization options.
* **New Characters & More:** Additional playable characters like Yoshi, Toad, Wario and more.
